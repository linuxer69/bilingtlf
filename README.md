# TLFHost - Specialized Billing & Support System


## Tech Stack
- PHP (OOP / PDO)
- MySQL / MariaDB
- Pure CSS3 & HTML5 (Inter Font)

## How to Install Locally
1. Clone the repository.
2. Import the database tables using the SQL schemas provided in the tasks.
3. Create a `config/db.php` file based on your local database credentials.
